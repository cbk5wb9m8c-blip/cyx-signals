# CYX.SIGNAL — 曹语轩 作品集

单文件个人作品集网站。以"信号调谐"为核心隐喻，入口动画模拟竖琴拨弦搜频的过程，进入后通过五段式玻璃推子导航浏览作品。

## 技术栈

纯静态 HTML/CSS/JS，无构建工具，无框架依赖。运行时依赖：

- **Web Audio API** — 所有音效（竖琴琴弦合成、页面过渡音、导航悬浮音）
- **Canvas 2D** — 琴弦物理动画
- **MediaPipe Hands** (CDN) — 摄像头手势识别（可选）
- **GSAP 3.12** (CDN) — 过渡动画
- **Font Awesome 6.5** (CDN) — 图标

## 启动

```bash
python3 -m http.server 8080
# 打开 http://localhost:8080/index.html
```

需要同目录下放置 `3dphoto.png` 作为个人照片背景。

## 入口动画流程

入口动画共 6 个阶段，模拟"拨弦搜频 → 锁定信号 → 身份显现"的调频仪式：

| 阶段 | 英文标识      | 描述                                             |
| ---- | ------------- | ------------------------------------------------ |
| -1   | CHOICE        | 选择交互方式（摄像头手势 / 鼠标触控）            |
| 0    | LINE          | 垂直脉冲线，1.5s 后过渡                          |
| 1    | STRINGS       | 30 根交互式竖琴弦，拨动以搜寻信号                |
| 2    | SIGNAL_LOCKED | 信号锁定，琴弦同步共振                           |
| 3    | PROJECTOR     | 放映机光束展开，煤尘粒子漂浮                     |
| 4    | PHOTO_REVEAL  | 照片以胶片负片显影方式呈现 → 白闪过渡进入主页面 |

### 核心机制：信号调谐

- 每根琴弦对应唯一频率（55Hz–1200Hz，30 根指数分布）
- 拨动新琴弦：信号强度 +0.09；重复拨动：+0.02
- 信号强度 ≥ 0.95 时触发锁定，进入后续阶段
- 信号强度条实时显示在页面顶部
- 拨动琴弦时产生闪光碎片（胶片划痕 / 黑胶波纹 / 关键词 / 轮廓线）

### 竖琴音色设计

30 根琴弦分为 4 组音色，使用 Karplus-Strong 延迟线 + 三角波振荡器混合合成，模拟竖琴拨弦质感：

| 组       | 琴弦范围 | 频率范围   | 音色特征                     | 衰减  |
| -------- | -------- | ---------- | ---------------------------- | ----- |
| Bass     | 0–7     | 55–150Hz   | 深沉共鸣，双延迟体共振，低频增强 | 5.0s  |
| Mid-low  | 8–14    | 150–300Hz  | 温暖歌唱感，峰值 EQ 谐波强调  | 3.5s  |
| Mid-high | 15–21   | 300–580Hz  | 明亮清透，带通滤波强调        | 2.5s  |
| Treble   | 22–29   | 580–1200Hz | 空灵铃音，梳状延迟微光，短衰减 | 2.0s  |

### 视觉区分

- **低音弦**（Bass）：2px 宽，琥珀色辉光（`#E8960C`）
- **中音弦**（Mid）：1px 宽，紫色辉光（`#7B5EA7`）
- **高音弦**（Treble）：0.5px 宽，青蓝色辉光（`#0FA89E`）

### 琴弦联动

- 拨动某根琴弦联动相邻 ±2 根琴弦振动
- 持续按压积累 tension，松手释放更强 pluck
- 指针跨弦移动时自动释放旧弦并开始新弦

## 交互方式

### 鼠标 / 触控（默认）

- 鼠标滑过或手指触摸琴弦区域即拨动
- 支持多点触控（每根手指独立追踪）
- 琴弦有阻力感：持续按压某根琴弦积累 tension，松手时释放更强 pluck

### 摄像头手势

- 使用 MediaPipe Hands 在本地识别手势（不上传图像）
- 食指指尖位置映射到琴弦
- 拇指与食指捏合 = 捏住琴弦积累 tension（阈值 0.08，比默认更灵敏）
- 松开捏合 = 拨弦释放
- 检测置信度下调至 0.5，跟踪灵敏度提高
- 信号锁定后自动关闭摄像头
- 按 `ESC` 可从摄像头模式切回鼠标模式
- 摄像头权限被拒时自动降级为鼠标模式

## 主导航：玻璃推子

主页面使用 5 个独立的竖向玻璃推子导航，每段有独立配色：

| 编号 | ID     | 名称 | 配色       | 装饰元素                   |
| ---- | ------ | ---- | ---------- | -------------------------- |
| 00   | SOURCE | 自我 | 紫色       | 频谱柱状条（8 根随机跳动） |
| 01   | CINEMA | 电影 | 青蓝 teal  | 胶片齿孔                   |
| 02   | GROOVE | 音乐 | 琥珀 amber | SVG 音频波形               |
| 03   | CUT    | 剪辑 | 卡其 khaki | 剪辑切点标记               |
| 04   | SIGNAL | 联系 | 蓝灰       | 无线电信号点阵             |

### 推子特性

- 激活推子：各频道独立配色发光，3D 厚度，推子填充电平升至 92%
- 悬浮推子：轻微上浮，推子帽上移
- 点击推子触发页面切换，伴随独立过渡动画
- 悬浮时播放对应频段的短促微音效

### 主页面元素

- **玻璃信息卡片**（左上角）：姓名 + 拼音 + 个人信息，呼吸式发光
- **有机连接层**（SVG）：音频曲线、漂浮花瓣、胶片齿孔边缘
- **电影时间线 HUD**：时间码 / SCENE 名称 / REC 指示灯 / 24 FPS
- **全幅照片背景**：仅在 SOURCE 页面显示，切换内容页时过渡为纯黑

## 音效设计

### 竖琴合成（Karplus-Strong + 三角波振荡器）

30 根琴弦采用噪声脉冲激励 + 三角波体音混合合成：

- 短噪声爆发（指数衰减包络，8ms）模拟拨弦瞬态
- 三角波振荡器提供清晰的音高基频
- 多级延迟线反馈回路模拟琴体共振
- 低音弦：双延迟回路 + 低频搁架增强
- 高音弦：梳状延迟 + 微失谐制造闪烁感

### 页面过渡音效

SoundEngine 类提供 5 种页面切换音效：

- **SOURCE**: 低频心跳脉冲
- **CINEMA**: 放映机嗡鸣 + 胶片划痕咔嗒声
- **GROOVE**: 黑胶噼啪声 + 唱针落碟声
- **CUT**: 打击切片循环
- **SIGNAL**: 上升频率扫描

### 导航悬浮音

推子导航悬浮时播放对应频段的 150ms 微音效，频率从 45Hz–1.5kHz 按频段递增。

## 视觉风格

- 深色背景（`#080808`），各频道独立配色

- 胶片颗粒噪点纹理

- 自定义滚动条样式
- 等宽字体体系
- 页面过渡使用 clip-path 动画（film wipe / vinyl reveal / wave sweep / signal rings）
- `prefers-reduced-motion` 响应支持
